Task 1.1.9.c.

Assume that \(A=\Z\) and that \(\sim\) is defined by:
\begin{equation*} a\sim b \text{ if and only if } a\equiv b \pmod n. \end{equation*}
Prove that if \(C_1\) and \(C_2\) are (any) two equivalence classes of \(\sim\) then there is a bijective function \(f \colon C_1\rightarrow C_2\text{.}\)
