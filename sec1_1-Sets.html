    Section 1.1 Sets

    We all have an intuitive idea of what a set is. Still, the concept of a set has given headaches to several mathematicians throughout the history of the subject. This is perhaps because the intuitive definition of a set is too simple, yet easily misunderstood and misused. Consider for instance the following set \(A\text{:}\)
    \begin{equation*} A = \{ x; \ x\notin A\} \end{equation*}
    This set gives rise to what is called Russell’s Paradox, named after Bertrand Russell (1872-1970), a Welsh mathematician and philosopher. It is easy to see that if an element belongs to \(A\) then it cannot belong to \(A\text{,}\) and if an element does not belong to \(A\text{,}\) then it must be in \(A\text{.}\) Quite a conundrum!
    Sets like these called for a formal development of set theory. The achievement of this task is mostly attributed to Georg Cantor (1845-1918), and it is said to have driven Cantor certifiably insane. Many of the deep problems and questions of set theory are beyond the scope of this course, although a few of them are disturbingly `obvious’. Let us say for example that we want to choose an even number larger than the number of atoms in the world. This is something we assume feasible but... how would you find such a number? In order to deal with such questions the ability to choose an arbitrary element with a certain characteristic out of a set must be assumed. We use this assumption whenever we say `let’, `choose’, etc. in constructing a mathematical argument.
    The assumption mentioned above is called the axiom of choice, and is an essential building block of most of modern mathematics. This axiom is equivalent, or closely related, to many other important assumptions we make every day when doing mathematics. We list here some of the more commonly used ones: Trichotomy (a real number is either positive, negative or zero), every vector space has a basis, the well-ordering principle (Axiom \ref{WOPaxiom} on page \pageref{WOPaxiom}), the Cartesian product (see Definition \ref{cartesianproddef}) of two non-empty sets is non-empty, etc.
    For this course we will not need much beyond what the following simple definition of a set will allow us to do.

    Definition 1.1.1.

    A set is a non-ordered collection of items. The cardinality of a set \(A\text{,}\) denoted \(|A|\text{,}\) is the number of elements in \(A\) (if \(A\) has finitely many elements) or \(\infty\) if the set contains infinitely many elements.

    Definition 1.1.2.

    1. A subset of a set \(A\) is a set which only contains elements that also belong to \(A\text{.}\) A subset of \(A\) that is not equal to \(A\) is called a proper subset of \(A\text{.}\)
    2. \(A\cup B\text{,}\) the union> of \(A\) and \(B\text{,}\) is the set that contains all the elements belonging to \(A\) or \(B\) (or both).
    3. \(A\cap B\text{,}\) the intersection of \(A\) and \(B\text{,}\) is the set that contains all the elements belonging to both \(A\) and \(B\text{.}\)
    4. \(A - B\text{,}\) the set difference of \(A\) by \(B\text{,}\) is the set that contains all the elements of \(A\) that do not belong to \(B\text{.}\) This set is sometimes denoted by \(A \setminus B\text{.}\)
    5. The Cartesian product of \(A\) and \(B\text{,}\) denoted \(A\times B\text{,}\) is the set
      \begin{equation*} A\times B = \{ (a,b); \ a\in A, \ b\in B \}. \end{equation*}

    Remark 1.1.3.

    The union and intersection of \(A\) and \(B\) can also be presented as
    \begin{equation*} A\cup B = \{ x ; \ x\in A \ \text{or} \ x\in B \} \end{equation*}
    \begin{equation*} A\cap B = \{ x ; \ x\in A \ \text{and} \ x\in B \} \end{equation*}
    Understanding how elements of sets interact, and how they can be grouped, classified, and structured, is the foundation of mathematics --- and algebra, in particular. To achieve this understanding we use relations and, in particular, functions.
    Although you might be used to thinking that a function is only something that looks like \(f(x) = \sin(x)\) or \(g(x) = e^x + x^2-3\text{,}\) you need to recall that a function is just a special type of a relation, and that a relation is nothing but a subset of a Cartesian product of two sets. We formalize these ideas in the following definition.

    Definition 1.1.4.

    Let \(A\) and \(B\) be non-empty sets.
    1. A relation \(R\) from \(A\) to \(B\) is a subset of \(A\times B\text{.}\) By convention, the statement that \((a,b) \in R\) is often written as \(R(a)=b\text{.}\) We say that \(b\) is an image of \(a\) under \(R\text{,}\) or that \(a\) is mapped, or assigned, to \(b\) under \(R\text{.}\)
    2. If \(F\) is a relation from \(A\) to \(B\) such that every element in \(A\) is assigned to exactly one image in \(B\text{,}\) then \(F\) is said to be a function, and we write \(F \colon A \to B\text{.}\)
    3. If \(A = B\text{,}\) so that \(R\) is a relation from \(A\) to \(A\text{,}\) we say that \(R\) is a relation on \(A\). In this case, if \(x\text{,}\) \(y \in A\) and \((x,y) \in R\) we may write \(aRb\text{,}\) or \(a \sim b\) when \(R\) is understood.
    In the particular case of a relation from \(A\) to \(A\text{,}\) relations might have special properties.

    Definition 1.1.5.

    Let \(\sim\) be a relation on a set \(A\text{.}\)
    1. We say \(\sim\) is reflexiveif \(a\sim a\) for all \(a\in A\text{;}\)
    2. \(\sim\) is symmetricif \(a\sim b\) necessarily implies that \(b\sim a\text{;}\)
    3. \(\sim\) is transitiveif whenever \(a\sim b\) and \(b\sim c\text{,}\) it follows that that \(a\sim c\text{.}\)
    If a relation \(\sim\) satisfies all three properties listed above, then \(\sim\) is said to be an equivalence relation on \(A\text{.}\)
