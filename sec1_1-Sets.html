    Section 1.1 Sets

    We all have an intuitive idea of what a set is. Still, the concept of a set has given headaches to several mathematicians throughout the history of the subject. This is perhaps because the intuitive definition of a set is too simple, yet easily misunderstood and misused. Consider for instance the following set \(A\text{:}\)
    \begin{equation*} A = \{ x; \ x\notin A\} \end{equation*}
    This set gives rise to what is called Russellâ€™s Paradox, named after Bertrand Russell (1872-1970), a Welsh mathematician and philosopher. It is easy to see that if an element belongs to \(A\) then it cannot belong to \(A\text{,}\) and if an element does not belong to \(A\text{,}\) then it must be in \(A\text{.}\) Quite a conundrum!
    Sets like these called for a formal development of set theory. The achievement of this task is mostly attributed to Georg Cantor (1845-1918), and it is said to have driven Cantor certifiably insane. Many of the deep problems and questions of set theory are beyond the scope of this course, although a few of them are disturbingly `obviousâ€™. Let us say for example that we want to choose an even number larger than the number of atoms in the world. This is something we assume feasible but... how would you find such a number? In order to deal with such questions the ability to choose an arbitrary element with a certain characteristic out of a set must be assumed. We use this assumption whenever we say `letâ€™, `chooseâ€™, etc. in constructing a mathematical argument.
    The assumption mentioned above is called the axiom of choice, and is an essential building block of most of modern mathematics. This axiom is equivalent, or closely related, to many other important assumptions we make every day when doing mathematics. We list here some of the more commonly used ones: Trichotomy (a real number is either positive, negative or zero), every vector space has a basis, the well-ordering principle (Axiom \ref{WOPaxiom} on page \pageref{WOPaxiom}), the Cartesian product (see Definition \ref{cartesianproddef}) of two non-empty sets is non-empty, etc.
    For this course we will not need much beyond what the following simple definition of a set will allow us to do.

    Definition 1.1.1.

    A set is a non-ordered collection of items. The cardinality of a set \(A\text{,}\) denoted \(|A|\text{,}\) is the number of elements in \(A\) (if \(A\) has finitely many elements) or \(\infty\) if the set contains infinitely many elements.

    Definition 1.1.2.

    1. A subset of a set \(A\) is a set which only contains elements that also belong to \(A\text{.}\) A subset of \(A\) that is not equal to \(A\) is called a proper subset of \(A\text{.}\)
    2. \(A\cup B\text{,}\) the union of \(A\) and \(B\text{,}\) is the set that contains all the elements belonging to \(A\) or \(B\) (or both).
    3. \(A\cap B\text{,}\) the intersection of \(A\) and \(B\text{,}\) is the set that contains all the elements belonging to both \(A\) and \(B\text{.}\)
    4. \(A - B\text{,}\) the set difference of \(A\) by \(B\text{,}\) is the set that contains all the elements of \(A\) that do not belong to \(B\text{.}\) This set is sometimes denoted by \(A \setminus B\text{.}\)
    5. The Cartesian product of \(A\) and \(B\text{,}\) denoted \(A\times B\text{,}\) is the set
      \begin{equation*} A\times B = \{ (a,b); \ a\in A, \ b\in B \}. \end{equation*}

    Remark 1.1.3.

    The union and intersection of \(A\) and \(B\) can also be presented as
    \begin{equation*} A\cup B = \{ x ; \ x\in A \ \text{or} \ x\in B \} \end{equation*}
    \begin{equation*} A\cap B = \{ x ; \ x\in A \ \text{and} \ x\in B \} \end{equation*}
    Understanding how elements of sets interact, and how they can be grouped, classified, and structured, is the foundation of mathematics and algebra, in particular. To achieve this understanding we use relations and, in particular, functions.
    Although you might be used to thinking that a function is only something that looks like \(f(x) = \sin(x)\) or \(g(x) = e^x + x^2-3\text{,}\) you need to recall that a function is just a special type of a relation, and that a relation is nothing but a subset of a Cartesian product of two sets. We formalize these ideas in the following definition.

    Definition 1.1.4.

    Let \(A\) and \(B\) be non-empty sets.
    1. A relation \(R\) from \(A\) to \(B\) is a subset of \(A\times B\text{.}\) By convention, the statement that \((a,b) \in R\) is often written as \(R(a)=b\text{.}\) We say that \(b\) is an image of \(a\) under \(R\text{,}\) or that \(a\) is mapped, or assigned, to \(b\) under \(R\text{.}\)
    2. If \(F\) is a relation from \(A\) to \(B\) such that every element in \(A\) is assigned to exactly one image in \(B\text{,}\) then \(F\) is said to be a function, and we write \(F \colon A \to B\text{.}\)
    3. If \(A = B\text{,}\) so that \(R\) is a relation from \(A\) to \(A\text{,}\) we say that \(R\) is a relation on \(A\). In this case, if \(x\text{,}\) \(y \in A\) and \((x,y) \in R\) we may write \(aRb\text{,}\) or \(a \sim b\) when \(R\) is understood.
    In the particular case of a relation from \(A\) to \(A\text{,}\) relations might have special properties.

    Definition 1.1.5.

    Let \(\sim\) be a relation on a set \(A\text{.}\)
    1. We say \(\sim\) is reflexive if \(a\sim a\) for all \(a\in A\text{;}\)
    2. \(\sim\) is symmetric if \(a\sim b\) necessarily implies that \(b\sim a\text{;}\)
    3. \(\sim\) is transitive if whenever \(a\sim b\) and \(b\sim c\text{,}\) it follows that that \(a\sim c\text{.}\)
    If a relation \(\sim\) satisfies all three properties listed above, then \(\sim\) is said to be an equivalence relation on \(A\text{.}\)

    Checkpoint 1.1.6. Some Equivalence Relations Exercises.

    Show that the following relations are equivalence relations on the indicated sets:

    (a)

    \(\sim\) defined on \(\Z\) by \(a\sim b\) if and only if, for a fixed \(n\in \Z\text{,}\) \(\gcd(a,n) =\gcd(b,n)\) (see Definition ...).

    (b)

    \(\sim\) defined on the set of all finite sets by \(A\sim B\) if and only if there exists a \index{Equivalence Relations} bijective function \(f:A\rightarrow B\)

    (c)

    defined on \(\R^3\) by \(P \sim Q\) if and only if \(P-Q\) is orthogonal to the vector \((1,1,1)\text{.}\)

    (d)

    \(\sim\) defined on \(\R\) by \(a\sim b\) if and only if \(|a| =|b|\text{.}\)

    (e)

    \(\sim\) defined on \(\Z\) by \(a\sim b\) if and only if both are even or both are odd.

    (f)

    \(\sim\) defined on the set of all differentiable functions \(\R \rightarrow \R\text{,}\) where \(f\sim g\) if and only if \(f'(x) =g'(x)\) for all \(x\in \R\text{.}\)

    (g)

    \(\sim\) defined on \(\Z\) by \(a\sim b\) if and only if, for a fixed integer \(n>1\text{,}\) \(a\) and \(b\) have the same remainder when divided by \(n\text{.}\)

    (h)

    \(\sim\) defined on the set of all differentiable functions \(\R \rightarrow \R\text{,}\) where \(f\sim g\) if and only if \(f'(x) =g'(x)\) for all \(x\in \R\text{.}\)

    (i)

    \(\sim\) defined on the set of all triangles by \(\triangle ABC \sim \triangle DEF\) if and only if the two triangles are congruent. Would this relation still be an equivalence relation if we replaced `congruentâ€™ by `similarâ€™ ?

    (j)

    \(\sim\) defined on the set of all functions \(f:A\rightarrow B\) by \(f\sim g\) if and only if \(f(a)=g(a)\) for some fixed \(a\in A\text{.}\)

    Example 1.1.7. Equivalence Relation Example.

    Let \(A\) be a set and \(f \colon A \rightarrow A\) a fixed bijective function. We set
    1. \(\displaystyle f^0=id\)
    2. \(f^n = \underbrace{f\circ f\circ \cdots \circ f}_{n \,\mbox {times}}\text{,}\) and
    3. \(f^{-n} = (f^n)^{-1}\text{,}\) for all \(n\in \N\text{,}\) where \((f^n)^{-1}\) is the inverse function of\(f^n\text{.}\)
    Consider the relation \(\sim\)on \(A\)by \(a\sim b\)if and only if \(f^k(a) =b\text{,}\) for some \(k\in \Z\text{.}\) We next show that \(\sim\) is an equivalence relation.
    Let \(x\text{,}\) \(y\text{,}\) \(z \in A\text{.}\)
    \(\sim\) is reflexive:
    In order to check reflexivity we need to prove that\(x\sim x\text{.}\) That is, we need to find a power\(k\) of\(f\) such that\(f^k(x)=x\text{,}\) but since\(f^0\) is the identity function then\(f^0(x) = x\text{.}\) It follows that\(x \sim x\text{.}\)
    \(\sim\) is symmetric:
    Suppose\(x \sim y\text{.}\) This means that we know that there exists \(k \in \Z\) such that \(f^k(x) = y\) (definition of the relation). In order to show that\(y \sim x\) we need to find a power of\(f\) such\(f^r(y)=x\text{.}\) Since\(f^k\) sends \(x\) to \(y\text{,}\) then the inverse function of \(f^k\) sends\(y\) back to \(x\) (the inverse function exists because \(f\) is bijective, and thus any composition of\(f\)â€™s is also bijective, in particular \(f^k\)). It follows that\((f^k)^{-1}(y)=x\text{.}\) But, according to the definition above \((f^k)^{-1} = f^{-k}\text{.}\) It follows that \(f^{-k}(y) = x\text{.}\) Since \(k\in \Z\) then so is \(-k\text{,}\) therefore \(y \sim x\text{.}\)
    \(\sim\) is transitive:
    Suppose \(x \sim y\) and \(y \sim z\text{.}\) This means that there exist \(k\text{,}\) \(l \in \Z\) such that \(f^k(x) = y\) and \(f^l(y) = z\text{.}\) Since
    \begin{equation*} f^{l}f^{k} = \left( \underbrace{f\circ f\circ \cdots \circ f}_{l \,\mbox {times}} \right) \circ \left( \underbrace{f\circ f\circ \cdots \circ f}_{k \,\mbox {times}} \right) = \underbrace{f\circ f\circ \cdots \circ f}_{k+l \,\mbox {times}} \end{equation*}
    then \(f^{k+l}(x) = f^{l}f^{k}(x) = f^{l}\left( f^{k}(x) \right)= f^{l}(y) = z\text{.}\) Since \(k,l\in \Z\) then \(k+l\in \Z\text{,}\) and thus \(x \sim z\text{.}\)

    Definition 1.1.8.

    Let \(R\) be an equivalence relation on a set \(A\text{.}\) For a fixed \(x\in A\text{,}\) we define the (equivalence) class of \(x\) as
    \begin{equation*} [x] =\{ b\in A ; \ xRb \} \end{equation*}

    (c)

    Assume that \(A=\Z\) and that \(\sim\) is defined by:
    \begin{equation*} a\sim b \text{ if and only if } a\equiv b \pmod n. \end{equation*}
    Prove that if \(C_1\) and \(C_2\) are (any) two equivalence classes of \(\sim\) then there is a bijective function \(f \colon C_1\rightarrow C_2\text{.}\)
    You may recognize the equivalence relation in Exercise CheckpointÂ 1.1.6, part TaskÂ 1.1.6.g, as congruence modulo \(n\text{.}\) (Compare Exercise CheckpointÂ 1.1.9 part TaskÂ 1.1.9.c). The equivalence classes in this case are often called congruence classes. The congruence class of \(0\) consists of all the multiples of \(n\text{,}\) since
    \begin{equation*} [0] = \{ b \in \Z ; \ 0 \sim b \} = \{ b \in \Z ; \ b \equiv 0 \pmod{n} \} = \{ b \in \Z ; \ n \mid b \} = \{ \ldots, -3n, -2n, -n, 0, n, 2n, 3n, 4n, \ldots\}. \end{equation*}
    Similarly, check for yourself that
    \begin{equation*} [1] = \{ b \in \Z ; \ b \equiv 1 \pmod{n} \} = \{ \ldots, -2n+1, -n+1, 1, n+1, 2n + 1, \ldots\}\text{.} \end{equation*}
    The set of all distinct equivalence classes of \(\sim\) is
    \begin{equation*} \{ [0], [1], [2], \ldots, [n-1] \}, \end{equation*}
    which we usually call \(\Z_n\text{.}\) See subsection \ref{zee-n-subsection}. Note that every integer belongs to exactly one of these equivalence classes. In other words, the equivalence classes of \(\sim\) form a partition of \(\Z\text{.}\)
